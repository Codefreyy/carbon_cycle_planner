## 简介

基于朋友和我的健身计划需求，开发了一个碳循环计划生成小程序，主要根据碳循环饮食减肥法，输入身体数据（身高、体重、年龄等）和健身需求（热量缺口、营养素分配、低碳高碳日模式、开始日期），输出饮食计划。
